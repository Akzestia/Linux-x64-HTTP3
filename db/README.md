<h3>Creating tables with run.scripts.sh</h3>


### Postgress
  - Linux
    ```
    chmod +x run.scripts.sh
    ./run.scripts.sh
    ```
  - Windows (Git Bash)
    ```
    chmod +x run.scripts.sh
    ./run.scripts.sh
    ```

### Scylla
  - Linux
    ```
    chmod +x run.scripts.cql.sh
    ./run.scripts.cql.sh
    ```
